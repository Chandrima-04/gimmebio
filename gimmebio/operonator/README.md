linked_read_operon_finder
===============================

version number: 0.0.1
author: David Danko

Overview
--------

Find operons in metagenomic linked read data.

Installation / Usage
--------------------

To install use pip:

    $ pip install operonator


Or clone the repo:

    $ git clone https://github.com/dcdanko/operonator.git
    $ python setup.py install
